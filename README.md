# ZYRO — Audio Brand Landing Page

Портфоліо-проект у рамках **Frontend-марафону** (травень 2025).

## Посилання

- **Живий сайт:** https://zyro-landing.netlify.app
- **GitHub:** https://github.com/HiljSV/zyro-portfolio

## Про проект

Лендінг для вигаданого аудіо-бренду **ZYRO** — навушники, колонки, аудіо-окуляри.

Побудований на базі FLS v4 (Чертоги Фрілансера) — Vite + HTML + SCSS + JavaScript.

## Стек

| Інструмент | Призначення              |
| ---------- | ------------------------ |
| Vite       | Збірка та dev-сервер     |
| SCSS (BEM) | Стилізація, адаптивність |
| Vanilla JS | Меню, бургер, скрол      |
| Netlify    | Хостинг, CI/CD           |

## Секції

- **Hero** — головний банер з CTA
- **Recommended** — рекомендовані товари (hover-ефект)
- **Gallery** — сітка категорій з асиметричним grid-layout (Figma-макет)
- **Why ZYRO** — переваги бренду

## Запуск локально

```bash
npm install
npm run dev
```

## Збірка

```bash
npm run build
# output → /dist
```
